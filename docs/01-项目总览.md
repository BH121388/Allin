1. 概述与架构
1.1 工具定位
一个本地运行的基金智能投资决策工具，集成五大核心功能模块，基于量化策略模型和多因子评分体系，为个人投资者提供从基金推荐、持仓管理、深度查询到板块分析的完整决策支持。

1.2 整体数据流
text
用户打开工具 → 自动拉取当日行情数据
                    ├── 模块一：多因子模型筛选 → 输出5只基金 + 评分
                    ├── 模块五：板块排行 + 今日要闻 → 全局参考
                    └── 用户操作入口
                         ├── 模块二：选择基金 → 输入金额 → 实时操作建议
                         ├── 模块三：输入基金号 → 深度分析 + 评分
                         └── 模块四：鼠标悬停 → 重仓股 + 板块 + 实时涨幅
2. 模块一：每日基金智能推荐（5只）
2.1 策略来源总结
本模块采用多因子打分法进行基金筛选和评分。该方法是当前公募和券商主流的量化选基框架，覆盖全市场400余个因子，融合估值水平、盈利质量、研发成长、市场情绪等核心指标。

2.2 筛选因子与权重
采用以下六维度因子体系，每只基金满分100分：

维度	权重	评分指标	说明
收益动量	25%	近1月/3月/6月/1年收益率 + 年内收益率	多周期表现综合评分
风险控制	20%	最大回撤、波动率、下行标准差、Calmar比率	回撤越小得分越高
风险调整收益	15%	夏普比率、索提诺比率、信息比率	衡量单位风险收益
基金经理能力	15%	任职回报、任期时长、择时与选股Alpha	超额收益来源评估
规模与流动性	15%	基金规模、日均成交额、机构持有比例	适中规模（5-50亿）最优
行业景气度匹配	10%	重仓行业是否处于当前景气方向	匹配高景气行业加分
2.3 筛选逻辑流程
text
1. 从全市场基金池中，剔除成立不足1年、规模不足5000万、暂停申购的基金
2. 按"收益动量"因子降序排列，取前100只
3. 对前100只逐一计算六维度综合得分
4. 按综合得分降序排列，取前5只
5. 输出展示：基金代码、名称、类型、综合评分（百分制）、各维度分项得分

3. 模块二：持仓管理与操作建议
3.1 操作信号决策框架
参考量化择时与仓位管理的研究框架，设计四级操作信号体系。

3.1.1 四大操作信号定义
信号	触发条件	建议操作比例
买入	综合评分≥80且技术面多头排列	单次投入可用资金的10%-20%
持有	60≤综合评分<80，趋势未破	维持现有仓位
减持	综合评分降至40-60，或破20日均线	减持现有仓位的30%-50%
清仓	综合评分<40，或破60日均线+基本面恶化	卖出全部持仓
3.1.2 择时增强规则（技术面）
当指数位于200日均线下方时，整体进入防御模式，策略应主动降低仓位或暂缓买入：

均线信号：收盘价 > MA20 > MA60 > MA200 → 多头排列，趋势强劲

RSI信号：RSI(14)在30-70区间正常，>80超买需警惕，<30超卖可关注

MACD信号：DIF上穿DEA为金叉看涨，DIF下穿DEA为死叉看跌

成交量信号：放量上涨确认趋势，缩量上涨需警惕

3.2 定投策略
若操作信号为“买入”，需进一步判断是否启用定投模式。

3.2.1 定投策略选择
策略类型	适用场景	核心逻辑
估值定投（PE百分位法）	指数基金、行业基金	PE百分位<30%时增加投入，30%-70%正常定投，>70%减少或停止
均线定投	宽基及行业ETF	价格低于均线时多投（如低于年线15%以上加仓），高于均线时少投
价值平均定投	目标导向型投资者	每月调整投入金额使市值增长固定，跌多补多、涨多补少
普通定投	长期储蓄型	固定金额、固定频率投入，纪律性最强
3.2.2 定投金额计算逻辑
text
IF 估值百分位 < 20%:
    定投倍数 = 1.5-2.0倍（加倍投入，估值极度低估）
ELIF 估值百分位 < 30%:
    定投倍数 = 1.2-1.5倍（适度加码，低估区间）
ELIF 估值百分位 < 70%:
    定投倍数 = 1.0倍（正常投入，合理估值区间）
ELIF 估值百分位 < 90%:
    定投倍数 = 0.5倍（减少投入，高估区间）
ELSE:
    定投倍数 = 0倍（停止定投，极度高估）[reference:7]
用户自定义基准金额：用户输入每月可用于定投的总资金，系统根据定投倍数计算本次实际投入金额。

3.3 止盈策略
当基金盈利时，触发以下止盈判断：

基金类型	目标年化收益率	止盈方式
偏股型基金	8%-15%	分3批止盈（1/3、1/3、1/3），每上涨5%触发一批
混合型/固收+	4%-6%	分批减仓，到达目标收益率后卖出50%，剩余随涨随卖
行业/主题ETF	15%-25%	动量止盈：持有至趋势转弱（跌破20日均线）再清仓
3.4 持仓评分体系（与推荐评分一致）
持仓基金评分采用与模块一相同的多因子体系，实时更新。具体因子权重参见 第7节评分体系权重总表。

4. 模块三：基金查询与评分
4.1 功能描述
用户输入基金代码（6位数字），系统从公开数据接口获取基金完整信息，进行多维度评分，并输出操作建议和详细分析理由。

4.2 输入与输出
输入：基金代码（如 012345）
输出：

基金基本信息：名称、类型、成立日期、规模、基金经理、管理费率

综合评分（百分制，各维度分项展示）

操作建议：买入/持有/减持/清仓（+ 买入时是否定投及金额建议）

详细分析理由（不少于200字的文字分析，涵盖：）

业绩归因（收益来源分析）

风险提示（最大回撤、波动率）

持仓风格（大盘/小盘、价值/成长）

行业配置（重仓行业及景气度判断）

与同类基金比较（同类排名百分比）

5. 模块四：重仓股透视与板块识别
5.1 功能描述
用户鼠标悬停在任意基金名称/代码上时，弹出浮窗展示该基金的重仓持股详情。

5.2 数据来源
每季度基金季报披露的前十大重仓股（可通过东方财富、天天基金公开API获取）

部分平台提供重仓股实时行情追踪功能

注意：2026年部分销售平台已下架实时估值功能，本工具需通过合规数据源获取

5.3 浮窗展示内容
展示项	说明
基金名称 + 代码	标题行
十大重仓股表格	股票代码、名称、占净值比例、当日涨跌幅
重仓股加权涨幅	根据持仓占比加权计算，辅助判断基金当日表现
所属板块识别	自动判断该基金的重仓行业，标注板块标签（如“科技-TMT”、“新能源”、“消费”、“医药”、“金融”等）
数据截止日期	季报截止日（如“数据截止2026-03-31”）
5.4 板块自动识别逻辑
text

    基金板块标签 = 前三大行业名称（如“电子+计算机+通信”）
ELSE：
    基金板块标签 = "均衡配置型"

7.3 综合评分计算公式
text
基金综合得分 = A×25 + B×20 + C×15 + D×15 + E×15 + F×10
各维度子项得分均归一化至0-100分，然后乘以权重累加。
