# 当前问题清单 — API限制（暂不处理）

## 问题：无法获取全量A股列表

### 理想的推荐流程
新浪批量API拉取5500只A股 → 过滤ST/亏损 → 预筛 → Top100完整评分 → 输出Top5

### 实际遇到的API限制

| API | 用途 | 状态 |
|-----|------|------|
| 新浪 vip.stock.finance.sina.com.cn | 批量股票列表（含PE/PB/市值） | HTTP 456 频率封禁，不可恢复 |
| 东方财富 push2.eastmoney.com | 批量股票列表 | HTTP 空响应（网络不通） |
| 东方财富 82.push2.eastmoney.com | 批量股票列表（CDN节点） | HTTP 空响应（网络不通） |
| 腾讯 qt.gtimg.cn | **单只**实时行情 | ✅ 稳定可用 |
| 新浪 hq.sinajs.cn | **单只**实时行情 | ✅ 稳定可用 |

### 结论
- 批量列表API在此网络环境全部不可用
- 单只查询API（腾讯/新浪）可用，但不支持批量
- 唯一可行方案：维护一个精选股票代码池，逐只通过腾讯API获取真实PE/PB/市值/价格

### 当前替代方案
精选列表（约130只，覆盖全部申万一级行业）作为代码池，
每只通过腾讯API获取真实数据后参与评分，全部不随机不抽样。
