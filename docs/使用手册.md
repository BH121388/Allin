## 2. 基金模块

### 2.1 每日智能推荐

**路径：** 导航栏 → 「推荐」

**功能：** 系统每日自动从全市场基金池中筛选评分最高的 5 只基金。

**筛选逻辑：**
1. 剔除成立不足1年、规模不足5000万、暂停申购的基金
2. 按收益动量因子降序取前 100 只
3. 对前 100 只进行六维评分（详见第4节）
4. 按综合得分降序输出 Top 5

**页面内容：**
- 每只基金卡片显示：代码、名称、类型、综合评分（百分制）、六维雷达图
- 评分等级标注（★★★★★~★）
- 可点击卡片查看详情或添加到持仓

**自动刷新时间：** 每个交易日 15:30

### 2.2 基金深度查询

**路径：** 导航栏 → 「查询」

**输入：** 6 位基金代码（如 `012345`）

**输出报告包含：**

| 模块 | 内容 |
|------|------|
| 基本信息 | 名称、类型、成立日期、规模、基金经理、任职年限、任职回报 |
| 综合评分 | 六维分项得分 + 总分（0-100） |
| 操作建议 | 买入/持有/减持/清仓 + 建议仓位比例 |
| 定投建议 | PE分位、定投倍数、策略说明（仅"买入"时显示） |
| 详细分析 | 200字以上文字分析（业绩归因、风险、持仓风格、行业配置、同类比较） |
| 风险指标 | 最大回撤、波动率、夏普比率、索提诺比率、Calmar比率、Alpha/Beta |
| 重仓股透视 | 十大重仓股表格（代码、名称、占净值比、当日涨跌幅） |
| 板块标签 | 自动识别重仓行业归属 |
| 同业比较 | 同类排名百分位、同类平均收益 |
| 净值走势 | 近期净值历史数据 |

### 2.3 持仓管理

**路径：** 导航栏 → 「持仓」

**功能：**

**添加基金：**
1. 点击「添加基金」
2. 输入基金代码和成本净值
3. 系统自动获取基金名称、类型等基本信息

**持仓列表：** 每只持仓基金显示：
- 代码、名称、类型
- 持仓成本 vs 当前净值
- 盈亏金额与收益率
- 交易信号标签（买入 / 持有 / 减持 / 清仓）
- **定投计算器**：点击可输入预算金额，基于 PE 分位计算本次投入金额
- **止盈评估**：根据持仓盈亏和止盈策略，给出操作建议

**数据导出：** 支持将持仓数据导出为文件。

**五大止盈策略：**
1. **移动止盈** — 从最高点回撤一定比例触发
2. **目标止盈** — 达到目标收益率分批卖出
3. **评分下降止盈** — 综合评分跌破阈值触发
4. **技术止损** — 跌破关键均线（20日/60日）清仓
5. **趋势止盈** — 趋势转弱时退出

### 2.4 市场概览

**路径：** 导航栏 → 「市场」

**页面包含三大区域：**

**板块排行表：**
| 列 | 说明 |
|----|------|
| 排名 | 按今日涨跌幅降序 |
| 板块名称 | 中信一级行业分类 |
| 今日涨跌幅 | 当日表现 |
| 近5日涨跌幅 | 短期趋势 |
| 主力资金净流入 | 资金态度 |
| 上涨/下跌家数比 | 板块内分化程度 |
| 驱动逻辑 | 涨跌原因一句话总结 |

自动标记：
- 涨幅前5（🟢）和跌幅前5（🔴）
- 🔥 连续3日上涨的持续性热点板块

**今日要闻：**
- 宏观经济数据、政策变动、地缘政治事件
- 每条标注利好/利空板块及理由
- 底部汇总整体影响方向与建议关注机会

**今日红利机会：**
- 自动识别当日有催化剂的板块
- 标注适合短线/中线/长线配置

### 2.5 基金筛选器

**路径：** 导航栏 → 「筛选」

**功能：** 多条件组合筛选基金

**筛选维度：**
- 基金类型（偏股混合/灵活配置/指数型等）
- 评分范围（最低分~最高分）
- 规模范围
- 收益率要求

输出符合条件的基金列表。

## 3. A股股票模块

### 3.1 仪表盘

**路径：** 导航栏 → 「仪表盘」

**功能：** A股模块总览入口，汇总展示：
- 今日推荐 Top 股票（快速预览）
- A股主要指数行情（上证/深证/创业板/科创50/沪深300）
- 恐贪指数（市场情绪）
- 市场温度计（估值温度）
- 热门板块概览

### 3.2 股票智能推荐

**路径：** 仪表盘内自动展示，或点击推荐区域跳转

**推荐管道（5步）：**
1. **全市场扫描** — 从 A 股 5000+ 只股票中获取列表
2. **初筛过滤** — 剔除 ST、\*ST、退市整理、流通市值<20亿、亏损股
3. **K线数据获取** — 拉取近 1 年日 K 线
4. **六维评分** — 逐只计算评分（详见第4节）
5. **精选 Top 10** — 按综合得分降序输出中盘成长股

**推荐结果：** 每只股票显示代码、名称、行业、市值、PE、六维评分及分项得分。

**自动刷新时间：** 每个交易日 15:30

### 3.3 股票深度查询

**路径：** 导航栏 → 「股查询」

**输入：** 股票代码（6 位数字，如 `600519`）或股票名称

**输出报告包含：**

| 模块 | 内容 |
|------|------|
| 基本信息 | 名称、代码、行业（申万一级/二级）、市值（流通/总市值）、上市日期 |
| 财务数据 | PE(TTM)、PB、ROE、营收增速、净利润增速、净利率 |
| 综合评分 | 六维分项得分 + 总分（0-100） |
| **五维预测** | 情绪30%+技术25%+资金20%+板块15%+日历10% → 买/不买 + 波动区间 |
| 八项技术指标 | MA（5/10/20/60/250日均线）、RSI(14)、MACD、BOLL、KDJ、OBV、趋势判断、OBV背离 |
| 操作建议 | 买入/持有/减持/清仓 + 建议仓位 |
| 详细分析 | 基本面+技术面+资金面综合文字分析 |
| 风险指标 | 最大回撤、波动率、夏普比率、Alpha/Beta |
| 同业比较 | 同行业排名百分位 |

### 3.4 股票持仓管理

**路径：** 导航栏 → 「股持仓」

**功能：** 与基金持仓类似，管理个人股票持仓：

- **CRUD 操作**：添加/查看/编辑/删除持仓股票
- **盈亏计算**：持仓成本 vs 实时价格
- **交易信号**：每只持仓股票显示当前操作建议
- **止盈止损**：四策略（移动止盈/目标止盈/评分下降/技术止损）
- **数据导出**：支持持仓数据导出

### 3.5 股票筛选器

**路径：** 导航栏 → 「股筛选」

**筛选条件：**

| 条件 | 说明 |
|------|------|
| 行业 | 申万一级行业（可选多个） |
| 市值范围 | 最小/最大流通市值（亿元） |
| PE 范围 | 市盈率区间 |
| ROE 最低 | 净资产收益率下限 |
| 营收增速 | 最低营收同比增长率 |
| 评分区间 | 六维评分范围 |
| 排序方式 | 按评分/市值/涨幅排序 |

输出符合所有条件的股票列表。

### 3.6 市场全景

**路径：** 导航栏 → 「股市场」

**功能：** A股市场全局概览：

- **主要指数行情**（上证/深证/创业板/科创50/沪深300）
- **恐贪指数**（A股市场短期情绪）
- **市场温度计**（估值温度：冰点/低估/常温/过热/沸腾）
- **热门板块榜单**（当日涨幅前10且有消息面的板块）
- **板块涨跌幅榜**（概念/行业板块涨幅/跌幅排行）
- **市场大事**（宏观、产业、个股重大事件解读）
- **大盘后市展望**（券商观点汇总）

### 3.7 股票对比

**路径：** 导航栏 → 「对比」

**功能：** 同时对比多只股票的六维评分：

- 输入 2~5 只股票代码
- 并排展示各维度的评分对比
- 雷达图叠加对比
- 关键财务指标对比表
- 综合排序输出优胜者

### 3.8 策略回测

**路径：** 导航栏 → 「回测」

**功能：** 验证评分策略的历史有效性

**回测流程：**
1. 选择回测时间范围（起始日期 ~ 结束日期）
2. 系统对历史评分数据进行模拟
3. 追踪模拟持仓收益曲线
4. 计算关键指标：

| 指标 | 说明 |
|------|------|
| 累计收益率 | 回测期间总收益 |
| 年化收益率 | 折算年化收益 |
| 最大回撤 | 期间最大回撤幅度 |
| 胜率 | 盈利交易占比 |
| Alpha | 超额收益（相对基准） |
| 夏普比率 | 风险调整后收益 |
| 策略有效性 | 综合评估等级 |

**输出：** 收益曲线图 + 指标汇总表 + 逐笔交易明细

## 4. 评分体系说明

### 4.1 基金六维评分（满分100）

| 维度 | 权重 | 评分指标 |
|------|------|----------|
| 收益动量 | 25% | 近1月/3月/6月/1年收益率、年内收益 |
| 风险控制 | 20% | 最大回撤、波动率、下行标准差、Calmar比率 |
| 风险调整收益 | 15% | 夏普比率、索提诺比率、信息比率 |
| 基金经理能力 | 15% | 任职回报、任期时长、选股Alpha、择时Alpha |
| 规模与流动性 | 15% | 基金规模、日均成交额、机构持有比例 |
| 行业景气度匹配 | 10% | 重仓行业是否处于当前景气方向 |

**综合得分 = 各维度归一化得分 × 权重之和**

### 4.2 股票六维评分（满分100）

| 维度 | 权重 | 评分指标 |
|------|------|----------|
| 收益动量 | 25% | 近1月/3月/6月/1年涨跌幅 |
| 风险控制 | 20% | 最大回撤、波动率、下行标准差 |
| 风险调整收益 | 15% | 夏普比率、索提诺比率 |
| 公司质量 | 15% | ROE、净利率、营收增速、净利润增速 |
| 估值性价比 | 15% | PE分位、PB分位、同行业估值对比 |
| 行业景气匹配 | 10% | 所属行业景气度、资金流向 |

### 4.3 评分等级

| 评分区间 | 等级 | 含义 |
|----------|------|------|
| ≥85 | ★★★★★ | 强烈推荐 |
| 75-84 | ★★★★ | 推荐 |
| 65-74 | ★★★ | 谨慎偏乐观 |
| 50-64 | ★★ | 观望 |
| <50 | ★ | 不推荐 |

## 5. 交易信号与操作指南

### 5.1 四大操作信号

| 信号 | 触发条件 | 建议操作比例 |
|------|----------|--------------|
| **买入** | 综合评分 ≥ 80 且技术面多头排列 | 单次投入可用资金的 10%-20% |
| **持有** | 60 ≤ 综合评分 < 80，趋势未破 | 维持现有仓位 |
| **减持** | 综合评分 40-60，或跌破20日均线 | 减持现有仓位的 30%-50% |
| **清仓** | 综合评分 < 40，或跌破60日均线+基本面恶化 | 卖出全部持仓 |

### 5.2 定投策略（仅"买入"时触发）

| PE分位 | 定投倍数 | 策略 |
|--------|----------|------|
| < 20% | 1.5-2.0x | 加倍投入，极度低估 |
| 20-30% | 1.2-1.5x | 适度加码，低估区间 |
| 30-70% | 1.0x | 正常投入，合理估值 |
| 70-90% | 0.5x | 减少投入，高估区间 |
| > 90% | 0x | 停止定投，极度高估 |

### 5.3 均线择时增强

| 信号 | 条件 |
|------|------|
| 多头排列 | 收盘价 > MA20 > MA60 > MA200 |
| RSI 正常 | RSI(14) 在 30-70 |
| RSI 超买 | > 80（警惕） |
| RSI 超卖 | < 30（关注） |
| MACD 金叉 | DIF 上穿 DEA（看涨） |
| MACD 死叉 | DIF 下穿 DEA（看跌） |

## 7. 常见问题

### Q: 为什么基金评分和昨天不一样？
A: 评分基于最新的净值数据和市场行情，每日收盘后数据更新，评分也会动态变化。

### Q: 重仓股数据总是最新的吗？
A: 重仓股明细来源于基金季报，每季度更新一次（1月、4月、7月、10月），季报截止日与实际披露日有约1个月的延迟。但重仓股的当日涨跌幅是实时数据。

### Q: 为什么要区分"基金"和"A股"两个模块？
A: 基金适合中长期配置，评分侧重经理能力和业绩稳定性；股票适合中短线操作，评分侧重技术面、估值和资金面。两者评分维度和策略不同。

### Q: 回测能保证未来收益吗？
A: 不能。回测基于历史数据验证策略在过去的表现，不构成未来收益保证。所有评分和信号仅供参考，不构成投资建议。

### Q: 交易信号出现了"买入"，我一定要买吗？
A: 评分和信号是辅助决策工具，最终决定应结合您的风险承受能力、资金状况和市场判断。建议开启定投模式分散风险。

### Q: 数据接口请求失败怎么办？
A: 所有数据接口都有降级（fallback）机制：东方财富不可用时自动切换新浪，MCP 服务不可用时使用本地缓存。如果所有数据源都失败，页面会显示相应的错误提示。
